# Security Policy

## Scope

Basidium is a hardware stress and evaluation tool intended for authorized lab use only. Security reports related to the tool itself (bugs that could cause unintended behavior, privilege escalation, unsafe defaults) are welcome.

Reports about using Basidium to attack systems you do not own are out of scope and will not be addressed.

## Reporting a Vulnerability

Please report security issues directly by email:

**stits@stits.org**

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested fix if you have one

Do not open a public GitHub issue for security vulnerabilities. You will receive a response within 5 business days.
